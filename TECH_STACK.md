# 🛠️ Technology Stack

Event Treasure is built for performance, scale, and extremely fluid UX. This stack ensures a premium developer and user experience.

## Core Infrastructure

- **Framework:** [Next.js (App Router)](https://nextjs.org/) - Utilized for hybrid SSR/client-side rendering and a robust routing architecture.
- **Database:** [MongoDB](https://www.mongodb.com/) - Chosen for flexible, scalable aggregation of unstructured AI-processed event schemas.
- **Package Management:** `npm`

## Frontend & UX

- **Styling:** [Tailwind CSS](https://tailwindcss.com/) - Integrated for rapid, consistent styling.
- **Motion:** [Framer Motion](https://www.framer.com/motion/) - Powers kinetic, Awwwards-inspired typography and smooth micro-interactions.
- **Layout:** [`masonic`](https://github.com/nanostores/masonic) - Provides high-performance, gapless algorithmic masonry grid rendering for the Bento Box aesthetic.
- **Icons:** [Lucide React](https://lucide.dev/) - Consistent, high-quality vector iconography.

## Data & Utilities

- **Date/Time:** [`dayjs`](https://dayjs.g_ip.io/) - Used for robust relative time formatting and local timezone rendering.
- **Calendar Exports:** [`ics`](https://www.npmjs.com/package/ics) - Enables client-side generation of RFC-compliant `.ics` files for native calendar integration.
- **Type Safety:** [TypeScript](https://www.typescriptlang.org/) - Ensures comprehensive type checking across the entire full-stack surface.
